Community standards

Built for safe, useful local groups.

Echo should help people find human connection without making the experience feel like a public performance.

Clear intent

Profiles focus on languages spoken, languages being learned, availability, and meetup preferences.

Public venues first

Meetup suggestions lean toward public, familiar places where new members can feel comfortable.

Warm introductions

Hosts can set simple prompts so new members know what to say when they arrive.

Healthy boundaries

Group context, report paths, and visible host details help keep early communities accountable.

For hosts

Start a local language circle without managing a giant group chat.

Hosts can create a focused meetup, describe the level, set expectations, and let nearby learners join when the fit is right.
